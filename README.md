# Overview
I developed a grid of sonic rings that rotates each ring to the y-cooridnate automatically.
An audio sound is also added to the scene. Whenever the user clicks anywhere on the page, the music "Green Hill Zone" starts playing. 
When you zoom into the ring, the sound gets louder but when you zoom out, it gets quieter.
It is used for practice and knowledge gained from Three.js journey.

# Installation
Want to run this project? Follow these steps:


1. Clone this repository into your computer:
  ```git clone https://github.com/schnattiba1/sonic-ring.git```


    Before continuing if you don't have Node.js, Vite, and Three.js installed, please check out the link below:

      Node.js: https://nodejs.org/en

      Vite: https://www.npmjs.com/package/vite

      Three.js: https://www.npmjs.com/package/three?activeTab=versions

   




2. Install node packages:
   
   
   ```npm install```
   
   
   You should see the "node_modules" directory.







3. Run the entire project:
   ```npx vite```
